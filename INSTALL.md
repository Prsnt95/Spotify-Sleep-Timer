## Troubleshooting

**"Load unpacked" button is grayed out?**
- Make sure Developer Mode is enabled (step 3)

**Extension disappeared after restart?**
- The extension folder was moved or deleted
- Reload it from its permanent location

**Timer doesn't work?**
- Make sure you're on `open.spotify.com` (not the desktop app)
- Refresh the Spotify page and try again
